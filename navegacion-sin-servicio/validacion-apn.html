Validación de APN

Lineamientos

  • Si APN está vacío, indicar al cliente que debe realizar unos pasos para configuración de APN desde el equipo
  • Actualizar el OV y ya debería aparecer el APN configurado
  • Comprueba con cliente si ya funciona el servicio
  • Si cliente quiere generar un RECLAMO, sigue el flujo regular [a_id/1819]

PANTALLA

  • 1. Revisar si el APN está vacío
  • 2. Guiar al cambio de red en caso sea necesario
    • 2.1. Ingresar al Link: https://www.helpsmartphone.com/public/es-ES/devices/
    • 2.2. Seleccionar la marca y modelo del cliente
    • 2.3. Seleccionar Redes
    • 2.4. Ingresar a Configurar Internet
    • 2.5. Seguir los pasos que se indican
  • 3. Actualizar operator view para validar el ingreso del APN
