DNS

Lineamientos

  • Validar con el cliente si le aparece el mensaje de DNS privado en el equipo
  • Ayuda al cliente para cambiar la configuración de DNS a automático
  • Si cliente quiere generar un RECLAMO, sigue el flujo regular, enlace

Pantallas y Validaciones

  • 1. Validar con el cliente si le aparece el siguiente mensaje en el equipo
  • 2. Guiar al cliente para configurar el DNS privado a automático
    • Andorid: Ajustes > Conexiones > Más ajustes de conexión > DNS
    • Iphone: Configuración > wifi > Seleccionar botón "i" (aparece al lado de la red a la que estamos conectados) > Configuración DNS
