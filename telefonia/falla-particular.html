Falla desde un número particular

Lineamientos

  • Identificar si no recibe llamadas de un solo número o de ningún número.
  • Si el problema es con un número Entel del mismo titular realizar los descartes al número, pero si no indicar que este número se contacte con nosotros para su revisión y realizar algunos descartes.
  • Si el problema es con un número de un solo operador se debe indicar que ese número debe contactarse con su operador para su revisión.
  • Si el problema es con todas las llamadas indicar que se derivará a otra área.
  • Si se desea enviar el caso a segundo nivel, recuerda seguir los pasos del siguiente ID https://informatepersonas.custhelp.com/app/answers/detail/a_id/6687
  • Si cliente quiere generar un RECLAMO, sigue el flujo regular https://informatepersonas.custhelp.com/app/answers/detail/a_id/1819
