Volte Homolagado

Lineamientos

  • SOLO CUANDO CLIENTE INDICA NAVEGAR CON LENTITUD MIENTRAS SE ENCUENTRA EN LLAMADA
  • Identificar si el cliente navega con lentitud mientras se encuentra en llamada.
  • Mientras está en llamada, validar en OV si la Ultima Red Registrada de Datos sale en 3G y la Disponibilidad de Voz sale: Ocupado para llamadas. Eso significa que no tiene habilitado el VOLTE, ayudar al cliente a activar el Volte https://www.entel.pe/incremento-4g/
  • Si el cliente se comunica por whatsapp, indicarle que realice una llamada y mantenga la comunicación por chat para ver su estado en OV
  • Si el Volte se encuentra activado, pero no aparece el icono significa que el modelo no está homologado o debe escalar a PTP
  • Si se desea enviar el caso a segundo nivel, recuerda seguir los pasos del siguiente ID https://informatepersonas.custhelp.com/app/answers/detail/a_id/5951
  • Si cliente quiere generar un RECLAMO, sigue el flujo regular https://informatepersonas.custhelp.com/app/answers/detail/a_id/1819

PANTALLA

  • Como detectar si el equipo del cliente tiene VOLTE activo en OV
  • Si el cliente no cuenta con el Volte activado guiarlo para la activación https://www.entel.pe/incremento-4g/
  • Verificar que el equipo se encuentre con Volte activado
  • Si el Volte se encuentra desactivado guiar al cliente a la activación
  • Si el Volte se encuentra activado en el equipo, pero no aparece el ícono en la pantalla derivar a Plataforma Técnica
