Configuracion de DNS

Lineamientos

  • SOLO CUANDO CLIENTE MENCIONA QUE LE FIGURA MENSAJE: “la red móvil no estaba disponible, no se puede acceder al servidor DNS privado”.
  • Validar con el cliente si en su equipo tiene configurado un DNS Privado
  • Ayuda al cliente para cambiar la configuración de DNS a automático
  • Si cliente quiere generar un RECLAMO, sigue el flujo regular https://informatepersonas.custhelp.com/app/answers/detail/a_id/1819

Pantallas y Validaciones

  • Validar con el cliente si en su equipo tiene configurado un DNS Privado
  • Guiar al cliente para configurar el DNS privado a automático
  • Andorid: Ajustes > Conexiones > Más ajustes de conexión > DNS
  • Iphone: Configuración > wifi > Seleccionar botón "i" aparece al lado de la red a la que estamos conectados > Configuración DNS
