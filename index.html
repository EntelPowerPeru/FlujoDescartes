Navegación - Sin servicio

Cuando no permite navegar en ninguna página o app. Aparece mensaje "Sin Internet" o "No hay conexión a Internet“. O, cuando no tenga ninguno de los 3 servicios: navegación, telefonía y SMS.

Navegación - Lentitud e Intermitencia

Las páginas web o aplicaciones se demoran en cargar.
Telefonía

Cuando el cliente reporte problemas con llamadas telefónicas entrantes o salientes.
Mensajería

Cuando el cliente reporte problemas con mensajes de texto entrantes o salientes.

Roaming

Cuando el cliente reporte problemas con el servicio roaming (datos o voz).
APP Ilimitadas

Cuando el cliente reporte problemas con aplicaciones ilimitadas.
