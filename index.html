Falla de navegación - Sin servicio
Cuando no permite navegar en ninguna página o app, aparece el mensaje "Sin Internet" o "No hay conexión a internet". O, cuando no tenga ninguno de los 3 servicios: navegación, telefonía y SMS.
Las páginas web o aplicaciones se demoran en cargar.
Cuando el cliente reporta problemas con llamadas telefónicas entrantes o salientes.
Cuando el cliente reporta problemas con mensajes de texto entrantes o salientes.
Cuando el cliente reporta problemas con el servicio roaming (datos o voz).
Cuando el cliente reporta problemas con aplicaciones ilimitadas.

En el paso 3 del flujo por lentitud y el paso 6 del flujo sin servicio, debemos colocar las coordenadas (latitud y longitud) del lugar donde el cliente presenta problemas de navegación. Estos datos podemos obtenerlos de Google Maps.
